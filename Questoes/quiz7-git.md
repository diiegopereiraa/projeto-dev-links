## Question 7/7
### O que são media queries no CSS?


`As media queries permitem adaptar a apresentação de um documento a diferentes dispositivos`

As media queries são uma técnica do HTML

Media queries no CSS são uma técnica obsoleta que não é mais usada para estilizar páginas da web.

Media queries no CSS são usadas para selecionar elementos HTML com base em suas características, como classe, ID ou tipo.
